## 💻 Install

Make sure you have [Node.js](http://nodejs.org/) and the [Expo](https://docs.expo.dev/) installed.

```sh
$ git clone https://github.com/Tyrone-Ward/donutAppUI.git
$ cd donutAppUI
$ npm install
```

<br>

## 📱 Usage

To start Expo run this line:

```sh
expo start
```
